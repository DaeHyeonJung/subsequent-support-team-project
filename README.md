## 프로젝트 개요
본 프로젝트는 항공·방산 분야의 후속지원 관점에서, 운용 중인 무인기 편대의 상태 변화에 대응하여  
운용자의 의사결정을 지원할 수 있는 소프트웨어를 설계하는 것을 목표로 한다.

특히 임무 수행 중 일부 기체의 이탈, 배터리 잔량 변화, 역할 변화, 탑재 장비 차이 등과 같은  
운용 환경의 변수를 반영하여, 남은 기체들로부터 임무 수행 가능한 편대 구성을 재산정하고  
적절한 편대 형태를 추천하는 로직을 구현하는 데 초점을 둔다.

본 프로젝트는 단순 제어 알고리즘 개발에 그치지 않고,  
후속지원 소프트웨어 설계, 운용자 판단 지원, 체계적 소프트웨어 개발 프로세스 적용을 함께 고려한다.

## 프로젝트 목표
- 무인기 편대 운용 상황에서 발생할 수 있는 기체 이탈 및 상태 변화 요소를 식별한다.
- 기체 수, 역할, 배터리 잔량, 탑재 장비 등의 조건을 반영한 편대 재구성 로직을 설계한다.
- 운용자가 임무 지속 여부와 편대 재편 방향을 판단할 수 있도록 의사결정 지원 기능을 제공한다.
- 항공·방산 소프트웨어 개발 관련 기준과 문서 체계를 참고하여 프로젝트 산출물을 체계적으로 관리한다.
- GitHub 기반 협업 환경에서 회의록, 설계 자료, 코드, 산출물을 구조적으로 관리한다.

### 최건희
- 결과 및 미팅 내용 정리
